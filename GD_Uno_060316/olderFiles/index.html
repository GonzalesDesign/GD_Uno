ES6 AND WEBPACK

Setting up ES6 with WebPack and Babel

NodeJS already installed.

Start by creating the project directory and setup the index.html.

Terminal:
npm init to create the package.json
Open package.json and edit info
Go back to Terminal
npm install : Goes and grabs the diff packages
Create webpack.config.js manually


To run Webpack Dev Server, copy and paste the following in the Terminal:
node_modules/.bin/webpack-dev-server
Test setup by running localhost:8080. Check Console for any errors.

To stop server: Ctrl C or In the Terminal window, left top click on "X" to close session.
